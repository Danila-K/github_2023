# Проекты

## E-comerce 
1. Выполнил анализ пользовательской активности:
- Категоризировал однократных покупателей для улучшения Retention Rate.
-	Исследовал причин недоставок заказов с учетом месячной динамики.
2. Оптимизация запасов и рекламных кампаний:
-	Проанализировал динамику покупок по дням недели для оптимизации рекламных стратегий.
3. Улучшение удержания и вовлеченности:
-	Провёл когортный анализ для выявления успешной когорты с высоким уровнем удержания.
4. RFM-сегментация и кластеризация:
-	Реализовал RFM сегментацию для понимания аудитории, учитывающей метрики ecency, frequency и monetary.
-	Кластеризация пользователей для более глубокого анализа поведения.

## A/B тестирование и анализ продукта
1. 	Анализ новой механики оплаты:
- Произвел A/B тестирование новой механики оплаты, анализ метрик (CR, ARPU, ARPPU) и статистической значимости.
-	Исследовал различия в показателях между контрольной и экспериментальной группами, выявил связи и сформулировал вывод о влиянии новой механики на поведение пользователей.
2. Автоматизация и визуализация:
-	Разработал функцию для автоматической загрузки данных из разных источников, включая дополнительные файлы, с последующим пересчетом метрик.
- Визуализировал полученные метрики в виде графиков и диаграмм для лучшего восприятия результатов и принятия решений.
